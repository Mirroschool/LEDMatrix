8  // This example shows how to cross-fade between different color palettes
9  // using the function nblendPaletteTowardPalette.
10  //
11  // The basic idea is that you always have a "current palette" that you're
12  // pulling colors from with ColorFromPalette, and you have a "target palette"
13  // which is the 'next' palette that you want to get to.
14  //
15  // After that, implementation is relatively simple: just periodically call
16  // nblendPaletteTowardPalette( currentPalette, targetPalette);
17  // If the current palette is not yet equal to the target palette, this
18  // function will make a few small changes to the current palette to make
19  // it slightly more like the target. Over time, the current palette will
20  // come to be equal to the target.
21  // There's no need to test the current and target for equality; it's safe
22  // to keep calling nblendPaletteTowardPalette even after current reaches target.
23  // For faster blending, call nblendPaletteTowardPalette twice per loop.
