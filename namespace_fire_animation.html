Detailed Description

MatrixFireFast - A fire simulation for NeoPixel (and other?) matrix displays on Arduino (or ESP8266) using FastLED.

◆ t

unsigned long FireAnimation::t = 0

make_fire() animates the fire display. It should be called from the loop periodically (at least as often as is required to maintain the configured refresh rate). Better to call it too often than not enough. It will not refresh faster than the configured rate. But if you don't call it frequently enough, the refresh rate may be lower than configured.
